Quick setup
Single command — paste into Terminal
System requirements
  • macOS 11.0+ (Big Sur or newer)
  • Apple Silicon · Intel-based Macs
  • git & bash 3.2+
Terminal guide
Launch Terminal
  1. Press Command (⌘) + Space to open Spotlight
  2. Type "Terminal" and press Return
Installation
  1. Copy the command from the field above
  2. Paste into Terminal and press Return
  3. Enter your password if prompted
  4. Restart your shell, or run source ~/.zshrc
  5. Verify with ragetree version
System-wide install (all users, requires admin):
curl -fsSL https://YOUR_USERNAME.github.io/ragetree/root-install.sh | sudo bash
